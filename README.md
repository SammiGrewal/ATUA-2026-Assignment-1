# ATUA-2026-Assignment-1

Analytic Exercise 1 - Spatial Regression & GWR

---

#### 🗺️ **Spatial Context**
- Using Aribnb listings data for Edinurgh, evaluate how price varies across the city and assess which factors best determine listing price.

---

#### 📈 **Regression Modelling**
The analysis proceeds through multiple spatial regression frameworks:

1. **OLS** – Baseline linear regression.
2. **Spatial Regression (ML/Error & ML/Lag, GM/Error & GM/Lag)** – Accounts for spatial dependence.
3. **Geographically Weighted Regression (GWR)** – Local coefficient variation.
4. **Multiscale GWR (MGWR)** – Local coefficient and bandwidth variation. 

Each model is fitted, evaluated, and mapped to compare performance and spatial patterns.

---

#### ✅ **Key Outcomes**
- Find the best predictors of price.
- Visualised how relationships between price and predictors vary across the city.
- Evaluate which model is the best for this data.
